# AI Powered Trading Signal Generator

## Overview

This project predicts Buy, Sell and Hold signals using Machine Learning.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- TA Library

## Features

- RSI
- MACD
- SMA20
- SMA50
- EMA20
- Random Forest Classifier
- Backtesting
- Interactive Dashboard

## Run

Install packages

```bash
pip install -r requirements.txt
```

Train model

```bash
python train.py
```

Launch dashboard

```bash
streamlit run app.py
```
